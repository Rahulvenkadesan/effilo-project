# 🛍️ Effilo - E-commerce Web Application

Effilo is a full-stack e-commerce web application developed using Django. It provides a modern UI and essential features for an online shopping experience.

---

## 🚀 Features

- User Registration & Login
- Product Listing & Categories
- Add to Cart Functionality
- Checkout Page
- Responsive UI Design

---

## 🛠️ Tech Stack

- Python
- Django
- HTML
- CSS
- Bootstrap

---

## 📁 Project Structure

- effilo_project/ - Main project settings
- shop/ - Core application
- templates/ - HTML files
- static/ - CSS, JS, images

---

## ⚙️ Installation

git clone https://github.com/Rahulvenkadesan/effilo-project.git  
cd effilo-project  
pip install -r requirements.txt  
python manage.py runserver
